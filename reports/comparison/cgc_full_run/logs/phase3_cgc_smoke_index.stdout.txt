
Welcome to CodeGraphContext!

CGC organises your code graphs using contexts:
  global    - One shared graph for all projects (default)
  per-repo  - Each repo gets its own .codegraphcontext/ folder
  named     - Create named workspaces (e.g. cgc index . --context MyProject)

Switch modes anytime:  cgc context mode <global|per-repo|named>
Or:                    cgc config set mode <global|per-repo|named>
