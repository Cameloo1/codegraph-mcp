CodeGraph Memory Layer CLI

Usage:
  codegraph-mcp <COMMAND> [ARGS]
  codegraph-mcp --help

Global flags:
  --repo <path>  --db <path>  --json  --no-color  --verbose  --quiet  --profile

Commands:
  init          Initialize CodeGraph state for a repository.
  index         Index a repository into the local graph store.
  status        Report local CodeGraph index status.
  query         Query symbols, text, files, references, definitions, calls, chains, or relation paths.
  impact        Report impact analysis for a file or symbol.
  context-pack  Build a compact proof-oriented context packet.
  context       Alias group for context packet generation.
  bundle        Export or import a CodeGraph bundle.
  watch         Watch a repository and incrementally re-index changed files.
  serve-mcp     Serve CodeGraph as a local MCP server.
  mcp           Alias group for serving the local MCP server.
  serve-ui      Serve the local proof-path UI.
  ui            Alias group for serving the local proof-path UI.
  bench         Run local reproducible CodeGraph benchmarks, including optional external CGC comparison.
  trace         Append replayable Agent/MCP JSONL trace events or replay/validate an events.jsonl file.
  audit         Run read-only audit inspections for storage, sampled edges, relation counts, and manual sample labels.
  languages     List language frontends, support tiers, exactness, and known limitations.
  doctor        Check local CodeGraph installation, index, optional resolver, MCP config, and UI assets.
  config        Print config, shell completions, or release/install metadata.

Phase 30: proof-focused UI/MCP ergonomics, real-repo corpus manifests, and honest parity reporting are available without weakening proof labels.
Workflow rule: no subagents; single-agent use only.
