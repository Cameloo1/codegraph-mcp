                                                                               
 Usage: cgc find [OPTIONS] COMMAND [ARGS]...                                   
                                                                               
 Find and search code elements                                                 
                                                                               
┌─ Options ───────────────────────────────────────────────────────────────────┐
│ --help          Show this message and exit.                                 │
└─────────────────────────────────────────────────────────────────────────────┘
┌─ Commands ──────────────────────────────────────────────────────────────────┐
│ name       Find code elements by exact name.                                │
│ pattern    Find code elements using substring matching.                     │
│ type       Find all elements of a specific type.                            │
│ variable   Find variables by name.                                          │
│ content    Search code content (source and docstrings) using full-text      │
│            index.                                                           │
│ decorator  Find functions with a specific decorator.                        │
│ argument   Find functions that take a specific argument/parameter.          │
└─────────────────────────────────────────────────────────────────────────────┘
